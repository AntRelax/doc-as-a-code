# Выбор внешней платформы

На платформе используется модуль сложного поиска с возможностью одновременного выбора нескольких фильтров. Данная функциональность может быть реализована с помощью внешней платформы.

Для этого подходит платформа Algolia. Записи предварительно загружаются на платформу. При учете, что потенциально количество задач не превышает 100К, кол-во записей умещается в базовый тариф Algolia. При поиске запрос отправляется в сервис, где происходит фильтрация и результат возвращается также через API. При увеличении нагрузки кол-во запросов будет выходить за рамки бесплатного варианта, после чего начнет тарифицироваться по 0.5 USD за 1К запросов (в месяц).

Использование внешней платформы обеспечивает высокую скорость доступа к данным и уменьшает сложность реализации проекта.

# Модель платформизации

Данный проект может быть использован образовательными организациями и онлайн школами для выдачи практических заданий, соответствующих экзаменационному формату и отслеживания статистики решения и аналитики по взаимодействию пользователей с платформой.

В качестве стратегии монетизации разумно выбрать формат подписки, стоимость которой будет скейлиться в зависимости от количество зарегистрированных на платформе пользователей.

Модель бизнеса - B2B2C. Услуга предоставляется не напрямую потребителю, а через посредника. Конкретно - проект используется образовательной организацией, которая предоставляет услугу школьникам и/или студентам.
