# Документация проекта **Платформа для подготовки ЕГЭ**

Репозиторий с документацией по личному проекту в формате docs-as-code.

## О проекте

Этот репозиторий содержит:
- бизнес-требования;
- модели бизнес-процессов (Event Storming, BPMN, UML);
- API-описание (OpenAPI);
- проектирование БД.

Основная идея: хранить всю проектную документацию в git и развивать ее вместе с проектом.

## Структура

```text
.
├── .github/
├── my-website/
│   ├── docs/                              # Документация (Markdown, схемы, спеки)
│   │   ├── requirements/                  # Требования
│   │   ├── BPmodel/                       # Модели бизнес-процессов
│   │   │   ├── EventStormingModel/
│   │   │   ├── BPMN/
│   │   │   └── UML/
│   │   ├── api/                           # REST API
│   │   ├── asyncapi/                      # Асинхронное взаимодействие
│   │   ├── BDdesign/                      # Проектирование БД
│   │   ├── platform/                      # Платформизация
│   │   ├── storage/                       # Хранение данных
│   │   └── layouts/                       # Макеты
├── .gitignore
└── README.md
```

## Статус

Документация в активной доработке.
